Space Game

## Descrição

Space Game é um jogo de nave vertical desenvolvido em JavaScript usando a biblioteca p5.js.
O jogador controla uma nave espacial, desvia de inimigos e dispara para eliminá-los, acumulando pontos e avançando de nível à medida que o score aumenta.

O jogo possui:

Menu de abertura com opção "Sobre" e exibição de recorde.

Sistema de vidas, score, nível e record.

Fundo com efeito parallax.

Explosões animadas e efeitos sonoros.

Progressão de dificuldade ao longo do tempo.

Controles

Setas → ↑ ↓ ← : Movimentar a nave.

Espaço : Atirar.

ENTER : Iniciar o jogo ou voltar ao menu.

S : Abrir tela “Sobre” no menu.

ESC : Voltar do “Sobre” para o menu.

 ## Como Jogar

Abra o index.html em um navegador compatível.

Pressione ENTER para iniciar.

Controle a nave, elimine inimigos e evite colisões.

Sobrevive com suas 3 vidas e tente bater o recorde!
